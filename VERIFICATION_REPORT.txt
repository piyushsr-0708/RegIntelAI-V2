AGENT DECISION LOGIC (from code inspection):
  
  Verdict Determination:
    • GO verdict:
      → machine_verifiable_checks > 0 AND manual_checks == 0
      → Fully automated control
      → Execute all automated checks
      
    • ESCALATE verdict (mixed automation):
      → machine_verifiable_checks > 0 AND manual_checks > 0
      → Execute automated portion THEN route manual checks for review
      → KEY: ESCALATE no longer blocks workflow
      
    • ESCALATE verdict (manual only):
      → machine_verifiable_checks == 0
      → Route all checks to compliance officer
      → No automated execution possible
  
  Confidence Derivation:
    • Primary: reasoned_control.confidence_metrics.overall_confidence
    • Secondary: verification_plan.confidence
    • Tertiary: automation_percentage
    • NOT using arbitrary fixed thresholds
  
  Reasoning Construction:
